Présentation

  • Language transcompilé en javascript
  • Developpée par microsoft, Anders Hejlsberg
  • Open source, code disponible sur github
  • Première version stable sortie en février 2012
  • Dernière version 2.0.5
  • Utilisée dans angular 2.0, nativescript...

Pourquoi typescript ?

  • Disposer des futures fonctionnalités du javascript (classes, decorators..)
  • Compilation vers es3, es5, es6
  • Typage statique
  • Facilité de réutilisabilité de code
  • Développement modulaire
  • Permet de faire des application sacalables
  • Un code javascript est un code typescript valide ( migration à coût 0)

Classes

        
    class User implements {
        firstName: string;
        age : number;
        constructor(firstName: string, age: number) {
            this.firstName = firstName;
            this.age = age;
        }

        render() {
            console.log(` fistName : ${this.firstName}, Age : ${this.age}.`);
        }
    }

Modules - Export

        
    export class User implements {
        firstName: string;
        age : number;
        constructor(firstName: string, age: number) {
            this.firstName = firstName;
            this.age = age;
        }
    }

Modules - Import

Decorators
